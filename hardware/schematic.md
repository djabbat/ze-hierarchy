# Схема — Prototype Bot

## Принципиальная схема

```
                    ┌──────────────────────┐
                    │      ESP32-C3         │
                    │                      │
                    │ GPIO1 ─┬── LED ── GND │
                    │        └── 220 Ом    │
                    │                      │
                    │ GPIO2 ──┬── Пьезо ── GND
                    │         └── 2N3904   │
                    │                      │
                    │ GPIO3 (ADC) ─┬── V_cond (через делитель)
                    │              │  └── 100к / 100к
                    │              │
                    │ GPIO4 ── Фотодиод ── GND
                    │         (через 10к pull-up)
                    │                      │
                    │ GPIO5 ── MAX9814 ── GND (микрофон)
                    │                      │
                    │ EN ── кнопка reset    │
                    └──────────────────────┘

                    ┌──────────────────────┐
                    │    RC-таймер (возраст)│
                    │                      │
                    │  C=1Ф ─── R=10МОм ── GND
                    │    │                  │
                    │    └────┬──── V_cond  │
                    │         │            │
                    │     LM393 (+)        │
                    │         │            │
                    │     V_ref (─)        │
                    │    от ESP32 2.5В     │
                    │         │            │
                    │     OUT ──┬── LED     │
                    │           └── Пьезо  │
                    └──────────────────────┘

                    ┌──────────────────────┐
                    │  Питание              │
                    │                      │
                    │ CR2032 ── 3.3V ── ESP32
                    │                      │
                    │ Li-Po ── 3.7V ── Мотор
                    │         │            │
                    │         └── 2N3904 ── GND
                    │              GPIO6   │
                    └──────────────────────┘
```

## Описание

### RC-таймер (возраст)
- C=1Ф заряжается до 5.0 В при производстве (подключить +5V на 1 сек)
- R=10МОм разряжает C: τ = 10⁷ с ≈ 115 дней
- V_cond(t) = 5.0 · exp(-t/115 дней)
- ADC ESP32 измеряет V_cond через делитель (100k+100k → 0-2.5В на ADC)

### Компаратор LM393
- (+) вход: V_cond
- (-) вход: V_ref = 2.5 В (от ESP32 через PWM+RC фильтр)
- OUT: пока V_cond > 2.5 В → LED + зуммер активны (молодой)
- OUT: когда V_cond < 2.5 В → LED + зуммер выключены (старый)

### Фототаксис
- Фотодиод (GPIO4): измеряет яркость окружающего света
- ESP32 читает ADC, определяет градиент
- Если ярко (рядом другой бот с LED) → корректирует направление мотора

### Звуковой репульсор
- MAX9814 (GPIO5): детектирует писк других ботов
- Если писк громче порога → бот отворачивает

### ESP-NOW
- Передаёт: ID (1Б) + V_cond_ADC (2Б) + возраст (2Б) + CRC (1Б) = 6Б
- Приём: внешний ESP32 на USB, опрос по TDMA

### Питание
- CR2032: только ESP32 (~1-3 мА) → > 200 часов
- Li-Po 100 мАч: только мотор (~50 мА) → ~2 часа
