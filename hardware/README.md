# Hardware — Ze-Hierarchy прототип

**Цель:** Собрать 1 работающий прототип бота для проверки концепции.

## Этапы

### Этап 1: Один бот на макетке
Проверить:
- RC-таймер (суперконденсатор 1Ф + 10МОм) — V_cond падает по экспоненте
- Компаратор LM393 — переключение LED + зуммера
- ESP32-C3 — передача ID по ESP-NOW
- Фотодиод MAX9814 — реакция на внешний свет

### Этап 2: Два бота
- Фототаксис: один яркий (новый) → второй (старый) движется к нему
- Звуковой репульсор: боты не сталкиваются
- ESP-NOW: приём ID на ПК

### Этап 3: 10 ботов
- Проверка ESP-NOW TDMA (опрос 10 устройств)
- Калибровка LM393 (подстройка порога)
- Измерение реального HI

## Контрольные точки

| Точка | Критерий | Ожидание |
|---|---|---|
| RC-таймер | V_cond измерен ADC ESP32 | V_0 = 5.0 В |
| Компаратор | LED + зуммер переключаются | Переход при 2.5 В |
| ESP-NOW | Пакеты принимаются | <5% потерь |
| Фототаксис | Старый движется к новому | HI > 0.5 |
| Репульсор | Боты не застревают | Нет столкновений >3 с |
