## Этап 1: Монтаж на макетке

### 1.1. RC-таймер (возраст)

```
Суперконденсатор 1Ф
  │                  │
(+)──────────────────┤
  │                  │
  └── R=10МОм ──────┘
  │                  │
  └── V_cond ────── ADC ESP32 (GPIO3) через делитель
                      │
                  100k ─── V_cond
                      │
                  100k ─── GND
                      │
                  ADC GPIO3
```

**Проверка:**
1. Зарядить C до 5.0V (подключить +5V на 1 сек)
2. Измерить V_cond мультиметром — должно быть 5.0V
3. Через 1 час: V_cond ~4.96V (падение 0.04V)
4. ADC ESP32: прочитать значение, пересчитать в V

### 1.2. Компаратор LM393

```
LM393 (DIP-8):
  ┌────┬────┐
1 │ OUT │ Vcc│ 8
  │     │    │
2 │ (-) │ (+)│ 7
  │     │    │
3 │ V+  │ (-)│ 6
  │     │    │
4 │ GND │ (+)│ 5
  └────┴────┘

Подключение:
  1 (OUT) → GPIO8 (для чтения) + LED через 220 Ом + пьезо через 2N3904
  2 (-) → V_ref = 2.5V (от ESP32 через PWM+RC фильтр)
  3 V+ → +3.3V
  4 GND → GND
  5 (+) → V_cond (через делитель)
  6 (-) → не используется
  7 (+) → не используется
  8 Vcc → +3.3V
```

**Настройка V_ref:**
```cpp
// PWM на GPIO9 → RC-фильтр → 2.5V
ledcSetup(2, 1000, 8); // 1 кГц, 8 бит
ledcAttachPin(9, 2);
ledcWrite(2, 194); // 194/255 * 3.3V ≈ 2.5V
```

### 1.3. LED + Пьезо

```
LM393 OUT ─┬── LED (анод)
           │   │
           │ 220 Ом
           │   │
           │   └── GND
           │
           └── 2N3904 (база)
                │
            коллектор ─── Пьезо (+) ─── +3.3V
```

### 1.4. Мотор

```
ESP32 GPIO6 ─── 1k ─── 2N3904 (база)
```

## Этап 3: Калибровка

### 3.1. ADC

1. Подать на GPIO3 известное напряжение (1.0V, 2.0V, 3.0V)
2. Записать ADC значение
3. Построить калибровочную кривую

```python
# calibration.py
import serial, numpy as np
ser = serial.Serial('/dev/ttyACM0', 115200)
v_known = [0, 1.0, 2.0, 3.0, 3.3]
adc_vals = []
for v in v_known:
    print(f"Set V_in = {v}V, then press Enter")
    input()
    ser.write(b'a')  # запрос ADC
    adc = int(ser.readline().strip())
    adc_vals.append(adc)
# Линейная регрессия
coeff = np.polyfit(adc_vals, v_known, 1)
print(f"V = {coeff[0]:.6f} * ADC + {coeff[1]:.6f}")
```

### 3.2. Компаратор

1. Измерить V_ref мультиметром (должен быть 2.5V)
2. Подать V_cond = 2.0V → LED должен быть выключен
3. Подать V_cond = 3.0V → LED должен гореть
4. Если не переключается — подстроить PWM V_ref
