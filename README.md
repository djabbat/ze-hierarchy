# Ze-Hierarchy

**Проверка эффекта возраста производства на коллективное поведение RNG-автоматов.**

Два поколения идентичных виброботов — новые (0 дней) и старые (30 дней) — в замкнутой арене. Новые ярче светят, старые следуют за светом. Звуковой репульсор не даёт сталкиваться.

| | |
|---|---|
| **Гипотеза** | Возраст производства влияет на поведение: интервенция (новые) vs мониторинг (старые) |
| **Метрика** | HI — Hierarchy Index |
| **N** | 120 ботов (2 × 60) |
| **Бюджет** | ~$566 |
