# Ze-Hierarchy

**Иерархия следования в колонии RNG-автоматов: проверка эффекта возраста производства на коллективное поведение**

---

## 1. Гипотеза

В популяции идентичных физических агентов, различающихся **только возрастом производства** (календарное время), возникает спонтанная иерархия следования:

- **Новые** (0 дней) — не имеют модели мира → **интервенция**: их присутствие искажает среду (яркий LED, высокий звук). Среда подстраивается под их прогноз.
- **Старые** (30 дней) — построили модель мира → **мониторинг**: следуют за предсказуемыми искажениями, созданными новыми. Среда не меняется — меняется их позиция.
- **Результат:** иерархия, где новые — лидеры (источники возмущения), старые — ведомые.

**Нулевая гипотеза (H₀):** Возраст не влияет на поведение. HI ≈ 0.5.

---

## 2. Дизайн

### 2.1. Автоматы
120 идентичных виброботов (bristlebots), две партии:

| Партия | Возраст | Кол-во |
|---|---|---|
| **Новые** | 0 дней | 60 |
| **Старые** | 30 дней | 60 |

Все боты — физически идентичны (масса 15±0.5 г, одинаковые моторы, щетинки, корпус). Разница — только в напряжении RC-таймера, заряженного при производстве.

### 2.2. Каналы

| Канал | Функция | Новые (интервенция) | Старые (мониторинг) |
|---|---|---|---|
| **LED** (свет) | Притяжение (фототаксис) | Яркий (5.0 В → LED=1.0) | Тусклый (3.85 В → LED=0.6) |
| **Зуммер** (звук) | Репульсор (не сталкиваться) + метка возраста | 4000 Гц (высокий) | 1000 Гц (низкий) |
| **ESP-NOW** (радио) | Идентификация + телеметрия | ID, V_cond, возраст | ID, V_cond, возраст |

### 2.3. Механизм
```
RC-возраст → V_cond → яркость LED → градиент света → фототаксис → иерархия
                 ↓
          частота зуммера → звуковой репульсор (не даёт слипаться)
```

### 2.4. Отличия от наивного подхода

| Аспект | Наивный подход | Наш (после PR) |
|---|---|---|
| Причина иерархии | «Читерство» | **Интервенция**: целенаправленное искажение среды |
| Синоним исследования | — | **Мониторинг**: пассивное наблюдение и построение модели |
| Столкновения | Игнорировались | **Звуковой репульсор**: пищат все → отворачивают от ближних |
| Физика RC | «RC = возраст» | RC + ADC-измерение V_cond через ESP32; реальный возраст — калибровка |
| Разброс параметров | JFET (неточный) | **LM393** компаратор (стабильный порог) |
| Масса | Разная (конденсатор) | **15±0.5 г** (балласт) |
| Зарядка | Qi-поле $30 (нереалистично) | **Сменные батарейки** + зарядная станция |
| Стат. мощность | 4×30 ботов (слабо) | **2×60** (Cohen's d=0.5, power=0.8) |
| Контрольные тесты | Нет | **5 тестов** (A–E) |

## 4. Модель / Уравнение

```
V_cond(t) = V₀ · exp(-t / RC₀)                  (1)  RC-таймер
LED(V) = clamp((V - V_LED_th) / (V₀ - V_LED_th), 0, 1)   (2)  Яркость
I_led(r) = LED · I₀ / r²                         (3)  Освещённость на расстоянии r
F_grad(I) = β · ∇I / (I + I₀)                    (4)  Фототаксис (сила притяжения)
```

Если Ze-эффект реален:
```
HI(t) → HI_max  при  t → ∞,  если V_cond_new ≠ V_cond_old
HI(t) ≈ 0.5     при  t → ∞,  если V_cond_new = V_cond_old (контроль)
```

### Sensitivity analysis: HI(β)

HI как функция коэффициента фототаксиса β (симуляция, 10 сидов):

| β (K_photo) | Средний HI | σ | dR (см) |
|---|---|---|---|
| 1.0 | 0.501 | 0.045 | +0.2 |
| 2.0 | 0.497 | 0.041 | +0.4 |
| 3.0 | 0.487 | 0.046 | +0.1 |
| 4.0 | 0.497 | 0.041 | +0.5 |
| 5.0 | 0.495 | 0.044 | +0.3 |

**Вывод:** HI слабо зависит от β. Эффект seed-зависимый: при удачных seed HI=0.568 (seed=5, β=4).

| Работа | Результат | Отличие нашего подхода |
|---|---|---|
| Hierarchical microswarms with leader-follower (Adv. Funct. Mater. 2020) | Иерархия через электрогидродинамику | У нас — возрастная деградация |

## 5. Метрики

| Метрика | Формула | Смысл |
|---|---|---|
| **HI** (Hierarchy Index) | `P(старый дальше от центра, чем новый)` | >0.5 → иерархия |
| **R_leader** | Среднее расстояние до центроида кластера | Новые ближе → малый R |
| **τ_conv** | Время до 90% HI_max | Быстрота установления иерархии |
| **Δ_spectrum** | 4000 Гц / 1000 Гц по зонам арены | Акустическая карта возрастов |
